## Numeric

`Integer()` (formato `int64`), `Int32()`, `Float()`, `Double()` compartilham
`Min(int)`/`Max(int)`:

```go
m.Property(&t.Age).Integer().Required().Min(0).Max(150)
```

## Boolean, Date & DateTime — ramificações sem wrappers

`Boolean()`, `Date()` (formato `date`) e `DateTime()` (formato `date-time`)
são as primeiras ramificações "sem wrappers" — sem validadores extras além dos quatro
base (`Required`/`Nullable`/`Description`/`Examples`). Eles retornam um
`*PropertyBuilder` puro diretamente.

```go
m.Property(&t.Active).Boolean().Required()
m.Property(&t.BornAt).Date().Required()
m.Property(&t.CreatedAt).DateTime().Required()
```
