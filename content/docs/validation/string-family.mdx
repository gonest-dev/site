All string-family branches share `Min(n)`, `Max(n)`, `Pattern(regex)`, and
`Enum(items ...string)`, and each sets an OpenAPI `format` value.

| Branch | OpenAPI `format` |
|---|---|
| `String()` | *(none)* |
| `Email()` | `email` |
| `Uuid()` | `uuid` |
| `Uri()` | `uri` |
| `Hostname()` | `hostname` |
| `Ipv4()` | `ipv4` |
| `Ipv6()` | `ipv6` |
| `Password()` | `password` |
| `Byte()` | `byte` |
| `Binary()` | `binary` |

```go
m.Property(&t.Name).String().Required().Min(1).Max(50)
m.Property(&t.Email).Email().Required()
m.Property(&t.Id).Uuid().Required()
m.Property(&t.Status).String().Required().Enum("active", "inactive")
```

`Enum` rejects any value outside the given list at request time (collected
alongside any other violation, never short-circuits) and emits `enum: [...]`
in generated OpenAPI docs.
