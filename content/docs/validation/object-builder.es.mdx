`Property(&t.Address).Object(func(om *gonest.ObjectMetadata) {...})` es de
**estado único (single-state)** — a diferencia de [Array](/docs/validation/array-builder), no hay
un concepto separado de "elemento". Todo el campo *es* el objeto.

## Reutilizando un schema registrado (`$ref`)

```go
m.Property(&t.Address).Object(func(om *gonest.ObjectMetadata) {
  om.Metadata(addressMetadata) // reuse an already-registered *Schema
  om.Required()
  om.Description("Endereço principal")
})
```

## Objetos abiertos / formato libre (free-form)

```go
m.Property(&t.Metadata).Object(func(om *gonest.ObjectMetadata) {
  om.AdditionalProperties() // e.g. for map[string]any
}).Nullable().Description("Metadados abertos do usuário")
```

`om.AdditionalProperties()` marca el schema como abierto, apropiado para
datos con forma de `map[string]any` sin un conjunto de claves fijo.
