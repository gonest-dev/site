---
title: Família de Strings
description: 10 ramificações de string compartilhando Min/Max/Pattern
---

Todas as ramificações da família de strings compartilham `Min(n)`, `Max(n)`, e `Pattern(regex)`, e
cada uma define um valor `format` no OpenAPI.

| Ramificação (Branch) | OpenAPI `format` |
|---|---|
| `String()` | *(nenhum)* |
| `Email()` | `email` |
| `Uuid()` | `uuid` |
| `Uri()` | `uri` |
| `Hostname()` | `hostname` |
| `Ipv4()` | `ipv4` |
| `Ipv6()` | `ipv6` |
| `Password()` | `password` |
| `Byte()` | `byte` |
| `Binary()` | `binary` |

```go
m.Property(&t.Name).String().Required().Min(1).Max(50)
m.Property(&t.Email).Email().Required()
m.Property(&t.Id).Uuid().Required()
```
