---
title: Validadores Personalizados
description: La válvula de escape Custom(fn)
---

`Custom(fn func(raw any) (any, error))` es una válvula de escape universal disponible
en cada `PropertyBuilder` — funciona de manera idéntica a través del cuerpo JSON, parámetros de ruta (path),
query string y campos de formulario.

```go
m.Property(&t.Slug).Custom(func(raw any) (any, error) {
  s, ok := raw.(string)
  if !ok || !slugPattern.MatchString(s) {
    return nil, fmt.Errorf("must be a URL-safe slug")
  }
  return s, nil
})
```

El error devuelto por `fn` se convierte en una violación en ese campo. Su valor devuelto
completa el campo de la estructura.

Usa `Custom` para formatos específicos de dominio o transformaciones que el vocabulario fijo
(String/Numeric/Boolean/DateTime/Array/Object) no puede expresar.

<Callout type="info">
  Esto absorbió la intención de una etapa de pipeline anterior, ahora eliminada, llamada
  `Pipe`/`NewPipe`. Si has visto `Pipe` mencionado en documentos antiguos de diseño de gonest,
  `Custom(fn)` es su reemplazo — `Pipe` ya no existe en la API actual.
</Callout>

<Callout type="warn">
  Limitación conocida: los campos `Custom(fn)` aparecen en los schemas OpenAPI generados
  con su name/required/nullable/description, pero **sin type o format** —
  un closure no tiene una forma declarativa de la cual inferir. Consulta
  [OpenAPI y Swagger](/docs/openapi).
</Callout>
