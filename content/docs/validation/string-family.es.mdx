title: Familia de Strings
description: 10 ramas de string que comparten Min/Max/Pattern
---

Todas las ramas de la familia de strings comparten `Min(n)`, `Max(n)` y `Pattern(regex)`, y
cada una establece un valor `format` en OpenAPI.

| Rama (Branch) | OpenAPI `format` |
|---|---|
| `String()` | *(ninguno)* |
| `Email()` | `email` |
| `Uuid()` | `uuid` |
| `Uri()` | `uri` |
| `Hostname()` | `hostname` |
| `Ipv4()` | `ipv4` |
| `Ipv6()` | `ipv6` |
| `Password()` | `password` |
| `Byte()` | `byte` |
| `Binary()` | `binary` |

```go
m.Property(&t.Name).String().Required().Min(1).Max(50)
m.Property(&t.Email).Email().Required()
m.Property(&t.Id).Uuid().Required()
```
