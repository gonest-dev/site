title: Object Builder
description: Campos de objeto aninhado de estado único
---

`Property(&t.Address).Object(func(om *gonest.ObjectMetadata) {...})` é
**estado único (single-state)** — diferente de [Array](/docs/validation/array-builder), não há
um conceito de "item" separado. O campo inteiro *é* o objeto.

## Reusando um schema registrado (`$ref`)

```go
m.Property(&t.Address).Object(func(om *gonest.ObjectMetadata) {
  om.Metadata(addressMetadata) // reuse an already-registered *Schema
  om.Required()
  om.Description("Endereço principal")
})
```

## Objetos abertos / forma livre (free-form)

```go
m.Property(&t.Metadata).Object(func(om *gonest.ObjectMetadata) {
  om.AdditionalProperties() // e.g. for map[string]any
}).Nullable().Description("Metadados abertos do usuário")
```

`om.AdditionalProperties()` marca o schema como aberto, apropriado para
dados no formato `map[string]any` sem um conjunto fixo de chaves.
