---
title: Validadores Customizados
description: A válvula de escape Custom(fn)
---

`Custom(fn func(raw any) (any, error))` é uma válvula de escape universal disponível
em todo `PropertyBuilder` — funciona de forma idêntica em corpo JSON, parâmetros de path,
query string e campos de formulário.

```go
m.Property(&t.Slug).Custom(func(raw any) (any, error) {
  s, ok := raw.(string)
  if !ok || !slugPattern.MatchString(s) {
    return nil, fmt.Errorf("must be a URL-safe slug")
  }
  return s, nil
})
```

O erro retornado por `fn` torna-se uma violação naquele campo. O seu valor retornado
preenche o campo do struct.

Use `Custom` para formatos específicos do domínio ou transformações que o vocabulário fixo
(String/Numeric/Boolean/DateTime/Array/Object) não pode expressar.

<Callout type="info">
  Isso absorveu a intenção de um estágio de pipeline anterior, agora removido, chamado
  `Pipe`/`NewPipe`. Se você já viu `Pipe` mencionado em documentos antigos de design do gonest,
  o `Custom(fn)` é o seu substituto — `Pipe` não existe mais na API atual.
</Callout>

<Callout type="warn">
  Limitação conhecida: campos `Custom(fn)` aparecem nos schemas OpenAPI gerados
  com seu nome/required/nullable/description, mas **sem type ou format** —
  uma closure não tem forma declarativa da qual inferir. Veja
  [OpenAPI & Swagger](/docs/openapi).
</Callout>
