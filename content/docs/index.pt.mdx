---
title: Documentação
description: gonest — um framework de injeção de dependência e HTTP para Go inspirado no NestJS
---

O **gonest** traz módulos, provedores, controladores e um pipeline de requisição completo
para o Go — o mesmo modelo mental que o NestJS oferece ao Node/TypeScript, sem abrir mão do Go idiomático.

<Callout type="warn">
  O gonest ainda não possui uma versão oficial lançada (`v0.x`) e não possui arquivo de `LICENSE`. Leia
  [Instalação](/docs/getting-started/installation) antes de depender dele em produção.
</Callout>
