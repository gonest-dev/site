title: Documentation
description: gonest — a NestJS-inspired dependency-injection and HTTP framework for Go
---

**gonest** brings modules, providers, controllers, and a full request
pipeline to Go — the same mental model NestJS gives Node/TypeScript, without
giving up idiomatic Go.

<Callout type="warn">
  gonest has no tagged release yet (`v0.x`) and no `LICENSE` file. Read
  [Installation](/docs/getting-started/installation) before depending on it
  in production.
</Callout>
