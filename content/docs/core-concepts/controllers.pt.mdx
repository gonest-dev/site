---
title: Controllers e Rotas
description: Agrupando rotas sob um prefixo de caminho
---

Um `Controller` agrupa rotas sob um prefixo de caminho. Construa um com
`gonest.NewController(func(controller *gonest.Controller) {...})`.

```go
var UserController = gonest.NewController(func(controller *gonest.Controller) {
  controller.Path("/users")
  controller.Tags("Users")
  userService := gonest.MustInject[*UserService](controller)

  controller.RouteGet("/", func(route *gonest.Route) {
    route.Handler(func(req *gonest.Request, res *gonest.Response) {
      res.Json(userService.List())
    })
  })
})
```

## Métodos do Controller

| Método | Descrição |
|---|---|
| `Path("/prefix")` | Define o prefixo de caminho para todas as rotas neste Controller. |
| `Tags("...")` | Tags do OpenAPI, herdadas por todas as rotas a menos que sobrescritas. |
| `BearerAuth()` | Marca todas as rotas como requerendo autenticação bearer no OpenAPI, a menos que sobrescritas. |
| `Use(...)` | Middleware no escopo do Controller. |
| `Guards(...)` | Guards no escopo do Controller. |
| `Interceptors(...)` | Interceptors no escopo do Controller. |
| `Filters(...)` | Filters de exceção no escopo do Controller. |

## Declarando rotas

`controller.Route(method, path, fn)` é a forma genérica. Atalhos por verbo
existem para todo método HTTP:

`RouteGet`, `RoutePost`, `RoutePut`, `RoutePatch`, `RouteDelete`, `RouteHead`,
`RouteOptions`, `RouteTrace`, `RouteConnect`, `RouteQuery`.

```go
controller.RouteGet("/:user_id", func(route *gonest.Route) {
  route.Summary("Get a user by id")
  route.Handler(func(req *gonest.Request, res *gonest.Response) {
    res.Json(map[string]string{"id": req.Param("user_id")})
  })
})
```
