---
title: Controllers y Rutas
description: Agrupando rutas bajo un prefijo de ruta
---

Un `Controller` agrupa rutas bajo un prefijo de ruta. Construye uno con
`gonest.NewController(func(controller *gonest.Controller) {...})`.

```go
var UserController = gonest.NewController(func(controller *gonest.Controller) {
  controller.Path("/users")
  controller.Tags("Users")
  userService := gonest.MustInject[*UserService](controller)

  controller.RouteGet("/", func(route *gonest.Route) {
    route.Handler(func(req *gonest.Request, res *gonest.Response) {
      res.Json(userService.List())
    })
  })
})
```

## Métodos de Controller

| Método | Descripción |
|---|---|
| `Path("/prefix")` | Establece el prefijo de ruta para todas las rutas en este Controller. |
| `Tags("...")` | Etiquetas de OpenAPI, heredadas por todas las rutas a menos que se anulen. |
| `BearerAuth()` | Marca todas las rutas como si requirieran autenticación bearer en OpenAPI, a menos que se anulen. |
| `Use(...)` | Middleware de ámbito de Controller. |
| `Guards(...)` | Guards de ámbito de Controller. |
| `Interceptors(...)` | Interceptors de ámbito de Controller. |
| `Filters(...)` | Filters de excepción de ámbito de Controller. |

## Declarando rutas

`controller.Route(method, path, fn)` es la forma genérica. Existen atajos por verbo
para cada método HTTP:

`RouteGet`, `RoutePost`, `RoutePut`, `RoutePatch`, `RouteDelete`, `RouteHead`,
`RouteOptions`, `RouteTrace`, `RouteConnect`, `RouteQuery`.

```go
controller.RouteGet("/:user_id", func(route *gonest.Route) {
  route.Summary("Get a user by id")
  route.Handler(func(req *gonest.Request, res *gonest.Response) {
    res.Json(map[string]string{"id": req.Param("user_id")})
  })
})
```
