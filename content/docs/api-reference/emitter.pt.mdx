`gonest.Emitter` — singleton do framework, que pode ser resolvido via `MustInject[*gonest.Emitter]` em qualquer Module.

| Símbolo | Descrição |
|---|---|
| `NewListener(func(listener *gonest.Listener) {...})` | Constrói um Listener. |
| `MustOn[EventType](listener, handler)` | Função livre (não um método) que registra um Handler tipado. |
| `Emitter.Emit(event)` | Dispara e esquece (fire-and-forget); uma goroutine por Listener; panics/erros são recuperados internamente. |

Sem eventos por padrão de string (wildcard/string-pattern) — os eventos são tipados por struct Go.

Veja: [Event Emitter](/docs/emitter)
