`gonest.NewGuard(func(guard *gonest.Guard) {...})`

| Method | Description |
|---|---|
| `Handler(func(req *gonest.Request, res *gonest.Response) bool)` | `false` produces an automatic 403; panicking with an `Exception` produces that exception's response. |

Applied via `Controller.Guards(...)`. Controller-scoped only — no `Module.Guards`, no `MustInject` inside the builder.

See: [Guards](/docs/request-pipeline/guards)
