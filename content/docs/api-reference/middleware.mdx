`gonest.NewMiddleware(func(middleware *gonest.Middleware) {...})`

| Method | Description |
|---|---|
| `Handler(func(req *gonest.Request, res *gonest.Response, next gonest.Next))` | `next(req, res)` continues the chain; omitting it short-circuits. |

Applied via `Module.Use(...)` (global, root only) or `Controller.Use(...)` (controller-scoped).

See: [Middleware](/docs/request-pipeline/middleware)
