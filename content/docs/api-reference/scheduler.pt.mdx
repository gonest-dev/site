`gonest.NewScheduler(func(scheduler *gonest.Scheduler) {...})`

| Método | Descrição |
|---|---|
| `Cron(name, expr, fn)` | Expressão cron de 5 campos (via `robfig/cron/v3`). |
| `Interval(name, dur, fn)` | Repete a cada intervalo (duration). |
| `Timeout(name, dur, fn)` | Dispara exatamente uma vez. |
| `Stop(name)` | Cancela uma tarefa (job) em execução. |

Cada execução de tarefa é isolada: possui seu próprio `recover`, sua própria goroutine.

Veja: [Scheduler](/docs/scheduler)
