`gonest.NewInterceptor(func(interceptor *gonest.Interceptor) {...})`

| Método | Descripción |
|---|---|
| `Handler(func(req *gonest.Request, res *gonest.Response, next gonest.InterceptorNext))` | El código antes de `next(req, res)` se ejecuta antes del Handler; el código después se ejecuta después. |

Aplicado a través de `Controller.Interceptors(...)`. Solo en el ámbito del Controller.

Consulta: [Interceptors](/docs/request-pipeline/interceptors)
