`gonest.NewInterceptor(func(interceptor *gonest.Interceptor) {...})`

| Método | Descrição |
|---|---|
| `Handler(func(req *gonest.Request, res *gonest.Response, next gonest.InterceptorNext))` | O código antes de `next(req, res)` é executado antes do Handler; o código depois é executado depois. |

Aplicado via `Controller.Interceptors(...)`. Apenas no escopo do Controller.

Veja: [Interceptors](/docs/request-pipeline/interceptors)
