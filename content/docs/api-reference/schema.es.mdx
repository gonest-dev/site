`gonest.NewSchema[T](func(t *T, m *gonest.Schema) {...})`

| Método | Descripción |
|---|---|
| `Title("...")` | Nombre en `components.schemas` (por defecto, el nombre del tipo Go). |
| `Property(&t.X)` | Identifica un campo por puntero, devuelve un `*PropertyBuilder`. |

## Restricciones base (todas las ramas)

`Required()`, `Nullable()`, `Description("...")`, `Examples(...)`

## Ramas (Branches)

| Rama | Devuelve | Notas |
|---|---|---|
| `String()`, `Email()`, `Uuid()`, `Uri()`, `Hostname()`, `Ipv4()`, `Ipv6()`, `Password()`, `Byte()`, `Binary()` | `*StringMetadata` | `Min`/`Max`/`Pattern` |
| `Integer()`, `Int32()`, `Float()`, `Double()` | `*NumericMetadata` | `Min`/`Max` |
| `Boolean()`, `Date()`, `DateTime()` | `*PropertyBuilder` | sin envoltorio (wrapper-less), solo restricciones base |
| `Array()` | `*ArrayMetadata` | estado dual (contenedor vs. elemento), consulta `.Items(fn)` |
| `Object(fn)` | `*ObjectMetadata` | estado único, `.Metadata(ref)` / `.AdditionalProperties()` |
| `Custom(fn func(raw any) (any, error))` | `*PropertyBuilder` | vía de escape universal (universal escape hatch) |

## Funciones de parsing

`gonest.MustParse[T](src, schema) T` / `gonest.Parse[T](src, schema) (T, error)`,
donde `src` es uno de `req.Params()`, `req.Query()`, `req.Headers()`,
`req.Body().Json()`, o `req.Body().Form(onFile)`.
