`gonest.NewController(func(controller *gonest.Controller) {...})`

| Método | Descripción |
|---|---|
| `Path("/prefix")` | Prefijo de ruta para todas las rutas en este Controller. |
| `Tags("...")` | Etiquetas de OpenAPI, heredadas por cada ruta a menos que se anulen. |
| `BearerAuth()` | Marca todas las rutas como si requirieran autenticación bearer, a menos que se anulen. |
| `Use(...)` | Middleware de ámbito de Controller. |
| `Guards(...)` | Guards de ámbito de Controller. |
| `Interceptors(...)` | Interceptors de ámbito de Controller. |
| `Filters(...)` | Filters de excepción de ámbito de Controller. |
| `Route(method, path, fn)` | Declaración genérica de ruta. |
| `RouteGet(path, fn)` / `RoutePost(path, fn)` / `RoutePut(path, fn)` / `RoutePatch(path, fn)` / `RouteDelete(path, fn)` / `RouteHead(path, fn)` / `RouteOptions(path, fn)` / `RouteTrace(path, fn)` / `RouteConnect(path, fn)` / `RouteQuery(path, fn)` | Atajos por verbo HTTP. |

## Métodos de `Route`

`Summary`, `Description`, `OperationId`, `Tags`, `BearerAuth`, `RequestBody(schema)`,
`Response(status, schema...)`, `PathParams(schema)`, `QueryParams(schema)`,
`ExcludeFromDocs()`, `Deprecated()`, `Params(schema)`, `FormBody(schema, fieldName)`,
`Handler(func(req *gonest.Request, res *gonest.Response) {...})`.
