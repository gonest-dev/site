`gonest.NewFilter(func(filter *gonest.Filter) {...})`

| Método | Descrição |
|---|---|
| `Catch(exemplar, handler)` | Corresponde pelo tipo concreto exato do valor do panic (`reflect.TypeOf`). |

Aplicado via `Controller.Filters(...)` ou `Module.Filters(...)` (global, apenas no raiz).
O nível do Controller tem precedência sobre o global para o mesmo tipo capturado. Envolve todo o pipeline.
