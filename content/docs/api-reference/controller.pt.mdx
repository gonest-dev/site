`gonest.NewController(func(controller *gonest.Controller) {...})`

| Método | Descrição |
|---|---|
| `Path("/prefix")` | Prefixo de caminho para todas as rotas neste Controller. |
| `Tags("...")` | Tags do OpenAPI, herdadas por todas as rotas, a menos que sobrescritas. |
| `BearerAuth()` | Marca todas as rotas como requerendo autenticação bearer, a menos que sobrescritas. |
| `Use(...)` | Middleware no escopo do Controller. |
| `Guards(...)` | Guards no escopo do Controller. |
| `Interceptors(...)` | Interceptors no escopo do Controller. |
| `Filters(...)` | Filters de exceção no escopo do Controller. |
| `Route(method, path, fn)` | Declaração genérica de rota. |
| `RouteGet` / `RoutePost` / `RoutePut` / `RoutePatch` / `RouteDelete` / `RouteHead` / `RouteOptions` / `RouteTrace` / `RouteConnect` / `RouteQuery(path, fn)` | Atalhos por verbo HTTP. |

## Métodos do `Route`

`Summary`, `Description`, `OperationId`, `Tags`, `BearerAuth`, `RequestBody(schema)`,
`Response(status, schema...)`, `PathParams(schema)`, `QueryParams(schema)`,
`ExcludeFromDocs()`, `Deprecated()`, `Params(schema)`, `FormBody(schema, fieldName)`,
`Handler(func(req *gonest.Request, res *gonest.Response) {...})`.

Veja: [Controllers & Routes](/docs/core-concepts/controllers), [OpenAPI & Swagger](/docs/openapi)
