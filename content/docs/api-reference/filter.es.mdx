`gonest.NewFilter(func(filter *gonest.Filter) {...})`

| Método | Descripción |
|---|---|
| `Catch(exemplar, handler)` | Coincide con el tipo concreto exacto del valor de panic (`reflect.TypeOf`). |

Aplicado a través de `Controller.Filters(...)` o `Module.Filters(...)` (global, solo en la raíz).
El nivel de Controller tiene precedencia sobre el global para el mismo tipo capturado. Envuelve todo el pipeline.

Consulta: [Filters](/docs/request-pipeline/filters)
