`gonest.NewInterceptor(func(interceptor *gonest.Interceptor) {...})`

| Method | Description |
|---|---|
| `Handler(func(req *gonest.Request, res *gonest.Response, next gonest.InterceptorNext))` | Code before `next(req, res)` runs before the handler; code after runs after. |

Applied via `Controller.Interceptors(...)`. Controller-scoped only.
