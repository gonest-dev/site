`gonest.NewMiddleware(func(middleware *gonest.Middleware) {...})`

| Método | Descrição |
|---|---|
| `Handler(func(req *gonest.Request, res *gonest.Response, next gonest.Next))` | `next(req, res)` continua a cadeia; omiti-lo causa curto-circuito (short-circuit). |

Aplicado via `Module.Use(...)` (global, apenas no raiz) ou `Controller.Use(...)` (escopo do Controller).

Veja: [Middleware](/docs/request-pipeline/middleware)
