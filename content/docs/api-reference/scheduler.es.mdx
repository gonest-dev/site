`gonest.NewScheduler(func(scheduler *gonest.Scheduler) {...})`

| Método | Descripción |
|---|---|
| `Cron(name, expr, fn)` | Expresión cron de 5 campos (vía `robfig/cron/v3`). |
| `Interval(name, dur, fn)` | Se repite en cada intervalo (duration). |
| `Timeout(name, dur, fn)` | Se dispara exactamente una vez. |
| `Stop(name)` | Cancela un trabajo (job) en ejecución. |

Cada ejecución de trabajo está aislada: tiene su propio `recover`, su propia goroutine.

Consulta: [Scheduler](/docs/scheduler)
