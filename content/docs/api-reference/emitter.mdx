`gonest.Emitter` — framework singleton, resolvable via `MustInject[*gonest.Emitter]` in any module.

| Symbol | Description |
|---|---|
| `NewListener[EventType](func(listener *gonest.Listener) func(context.Context, EventType) {...})` | Build a listener bound to `EventType`; the builder resolves dependencies once and returns the actual per-event handler. |
| `Emitter.Emit(event)` | Fire-and-forget; one goroutine per listener; panics/errors recovered internally. |

No wildcard/string-pattern events — events are typed by Go struct.

See: [Emitter](/docs/emitter)
