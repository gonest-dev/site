`gonest.Emitter` — framework singleton, resolvable via `MustInject[*gonest.Emitter]` in any module.

| Symbol | Description |
|---|---|
| `NewListener[EventType](func(listener *gonest.Listener[EventType]) {...})` | Build a listener bound to `EventType`; the builder resolves dependencies once, then calls `listener.On(handler)`. |
| `(*Listener[EventType]).On(func(ctx context.Context, event EventType))` | Register the per-event handler on a `Listener[EventType]`. |
| `Emitter.Emit(event)` | Fire-and-forget; one goroutine per listener; panics/errors recovered internally. |

No wildcard/string-pattern events — events are typed by Go struct.

See: [Emitter](/docs/emitter)
