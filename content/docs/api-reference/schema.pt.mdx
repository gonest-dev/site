`gonest.NewSchema[T](func(t *T, s *gonest.Schema) {...})`

| Método | Descrição |
|---|---|
| `Title("...")` | Nome em `components.schemas` (por padrão, o nome do tipo Go). |
| `Property(&t.X)` | Identifica um campo por ponteiro, retorna um `*PropertyBuilder`. |

## Restrições base (todos os ramos)

`Required()`, `Nullable()`, `Description("...")`, `Examples(...)`

## Ramos (Branches)

| Ramo | Retorna | Notas |
|---|---|---|
| `String()`, `Email()`, `Uuid()`, `Uri()`, `Hostname()`, `Ipv4()`, `Ipv6()`, `Password()`, `Byte()`, `Binary()` | `*StringMetadata` | `Min`/`Max`/`Pattern` |
| `Integer()`, `Int32()`, `Float()`, `Double()` | `*NumericMetadata` | `Min`/`Max` |
| `Boolean()`, `Date()`, `DateTime()` | `*PropertyBuilder` | sem wrapper (wrapper-less), apenas restrições base |
| `Array()` | `*ArrayMetadata` | estado duplo (contêiner vs. item), veja `.Items(fn)` |
| `Object(fn)` | `*ObjectMetadata` | estado único, `.Metadata(ref)` / `.AdditionalProperties()` |
| `Custom(fn func(raw any) (any, error))` | `*PropertyBuilder` | saída de escape universal (universal escape hatch) |

## Funções de parsing

`gonest.MustParse[T](src, schema) T` / `gonest.Parse[T](src, schema) (T, error)`,
onde `src` é um de `req.Params()`, `req.Query()`, `req.Headers()`,
`req.Body().Json()`, ou `req.Body().Form(onFile)`.
