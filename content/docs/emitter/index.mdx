---
title: Emitter
description: Typed, fire-and-forget events between providers
---

`gonest.Emitter` is a **framework singleton** — always resolvable via
`MustInject[*gonest.Emitter]` in any module, with zero explicit registration.

```go
type UserCreatedEvent struct{ UserID int64 }

var UserCreatedListener = gonest.NewListener(func(listener *gonest.Listener[UserCreatedEvent]) {
  listener.On(func(ctx context.Context, event UserCreatedEvent) {
    // handle the event
  })
})

var UserModule = gonest.NewModule(func(module *gonest.Module) {
  module.Listeners(UserCreatedListener)
})

var UserProvider = gonest.NewProvider(func(provider *gonest.Provider) {
  emitter := gonest.MustInject[*gonest.Emitter](provider)
  provider.Constructor(func() *UserService { return &UserService{emitter: emitter} })
})
```

<Callout type="info">
  `Listener[EventType]` is a real generic type — `NewListener`'s type argument is inferred from the
  builder function's own parameter. The builder receives the `*Listener[EventType]` itself (for
  `MustInject`/`MustInjectAll`) and is expected to call `listener.On(handler)` exactly once: any
  dependency is resolved ONCE, at bootstrap time, then closed over for every future event.
</Callout>

## Emit semantics

`Emitter.Emit(event)` is **fire-and-forget**: one goroutine per registered
listener for that exact event type, returns immediately, never blocks the
caller. A listener's panic or error is recovered internally and never
propagates.

There is **no wildcard/string-pattern** event matching — events are typed by
Go struct.

## Dynamic subscriptions (`Subscribe[T]`)

`NewListener`+`On` registers a static, app-lifetime handler. `gonest.Subscribe[T](emitter, done)` is
its dynamic counterpart — a channel that lives only as long as `done` stays open (typically one
connection's lifetime), closed automatically once it fires. This is what GraphQL Subscriptions
(see [GraphQL](/docs/graphql)) are built on:

```go
func(ctx *gonest.GraphqlContext, emit func(any)) {
  for event := range gonest.Subscribe[OrderCreatedEvent](emitter, ctx.Done()) {
    emit(event)
  }
}
```
