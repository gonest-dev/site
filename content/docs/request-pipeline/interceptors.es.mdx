---
title: Interceptors
description: Envoltura antes/después del handler al estilo AOP
---

Un `Interceptor` envuelve el Handler al estilo AOP. Construye uno con
`gonest.NewInterceptor(func(interceptor *gonest.Interceptor) {...})`.

```go
var TimingInterceptor = gonest.NewInterceptor(func(interceptor *gonest.Interceptor) {
  interceptor.Handler(func(req *gonest.Request, res *gonest.Response, next gonest.InterceptorNext) {
    // code here runs BEFORE the handler
    next(req, res)
    // code here runs AFTER the handler
  })
})
```

Los Interceptors se ejecutan después de los Guards y envuelven al Handler más todo lo anidado en
él. Son **solo del ámbito de Controller**.

## Aplicando un interceptor

```go
var UserController = gonest.NewController(func(controller *gonest.Controller) {
  controller.Interceptors(TimingInterceptor)
})
```
